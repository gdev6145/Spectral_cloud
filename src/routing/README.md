# Routing Engine

Intelligent packet routing with failover and optimization.

## Routing Algorithms

- **AODV** - Ad-hoc On-Demand Distance Vector
- **Babel** - Loop-avoiding distance-vector routing
- **Custom Optimization** - Latency and reliability aware

## Features

- Automatic route discovery
- Link quality monitoring
- Failover handling
- Latency optimization
- Load balancing
